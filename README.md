# bits

Go utilities and a [Claude Code plugin](https://code.claude.com/docs/en/plugins) marketplace.

## Install

```
/plugin marketplace add amwolff/bits
/plugin install amwolff-grimoire@bits
/plugin install chrome-devtools@bits
```

Also recommended: `/plugin install gopls-lsp@claude-plugins-official` (requires [`gopls`](https://pkg.go.dev/golang.org/x/tools/gopls)).
